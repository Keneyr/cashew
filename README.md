## Build
### With command line(Mac OS)
```shell
./configure && make
xctool -project Xcode/cashew/cashew.xcodeproj -scheme cashew build;
```
### With Xcode
1. ```./configure && make```
2. Open Xcode/cashew/cashew.xcodeproj with Xcode
